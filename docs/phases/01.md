# Phase 1: Core Parser

**Duration**: 2-3 weeks  
**Priority**: P0 Critical  
**Dependencies**: Phase 0  
**Goal**: Full ANSI/VT parser with conformance coverage ready for VTtest once a runnable terminal binary exists

---

## Overview

This phase implements the ANSI/VT escape sequence parser. The parser converts raw byte streams from the PTY into terminal operations (cursor movement, text attributes, screen manipulation, etc.). This is the heart of the terminal emulator - correctness and performance are critical.

### parser/state.rs

```rust
use super::actions::Action;
use crate::error::Result;

/// Parser state machine states.
#[derive(Clone, Copy, Debug, PartialEq, Eq)]
pub enum ParserState {
    /// Ground state - normal character processing.
    Ground,
    /// Escape state - saw ESC, waiting for next byte.
    Escape,
    /// Escape intermediate state - saw ESC followed by intermediate bytes.
    EscapeIntermediate,
    /// CSI entry state - saw ESC [.
    CsiEntry,
    /// CSI parameter state - collecting parameters.
    CsiParam,
    /// CSI intermediate state - collecting intermediates.
    CsiIntermediate,
    /// OSC string state - collecting OSC string.
    OscString,
    /// DCS entry state - saw ESC P.
    DcsEntry,
    /// DCS parameter state - collecting DCS parameters.
    DcsParam,
    /// DCS string state - collecting DCS string data.
    DcsString,
    /// SOS/PM/APC string state.
    String,
}

/// The ANSI/VT parser.
pub struct Parser {
    /// Current state.
    state: ParserState,
    /// Configuration.
    config: ParserConfig,
    /// Collected parameters.
    params: Vec<i64>,
    /// Collected intermediates.
    intermediates: Vec<u8>,
    /// Collected string data.
    string_data: Vec<u8>,
    /// Current character set (G0-G3).
    charset: [Charset; 4],
    /// Active character set.
    active_charset: usize,
    /// Saved cursor position.
    saved_cursor: Option<CursorPosition>,
}

/// Character set designations.
#[derive(Clone, Copy, Debug, PartialEq, Eq)]
pub enum Charset {
    /// US ASCII.
    Ascii,
    /// DEC Special Graphics.
    DecSpecial,
    /// UK character set.
    Uk,
    /// User-defined.
    User,
}

impl Default for Charset {
    fn default() -> Self {
        Self::Ascii
    }
}

impl Parser {
    /// Create a new parser.
    pub fn new() -> Self {
        Self {
            state: ParserState::Ground,
            config: ParserConfig::default(),
            params: Vec::with_capacity(16),
            intermediates: Vec::with_capacity(4),
            string_data: Vec::new(),
            charset: [Charset::Ascii; 4],
            active_charset: 0,
            saved_cursor: None,
        }
    }

    /// Create a parser with custom configuration.
    pub fn with_config(config: ParserConfig) -> Self {
        Self {
            state: ParserState::Ground,
            config,
            params: Vec::with_capacity(16),
            intermediates: Vec::with_capacity(4),
            string_data: Vec::new(),
            charset: [Charset::Ascii; 4],
            active_charset: 0,
            saved_cursor: None,
        }
    }

    /// Parse a byte and return actions.
    pub fn parse_byte(&mut self, byte: u8) -> Vec<Action> {
        match self.state {
            ParserState::Ground => self.parse_ground(byte),
            ParserState::Escape => self.parse_escape(byte),
            ParserState::EscapeIntermediate => self.parse_escape_intermediate(byte),
            ParserState::CsiEntry => self.parse_csi_entry(byte),
            ParserState::CsiParam => self.parse_csi_param(byte),
            ParserState::CsiIntermediate => self.parse_csi_intermediate(byte),
            ParserState::OscString => self.parse_osc_string(byte),
            ParserState::DcsEntry => self.parse_dcs_entry(byte),
            ParserState::DcsParam => self.parse_dcs_param(byte),
            ParserState::DcsString => self.parse_dcs_string(byte),
            ParserState::String => self.parse_string(byte),
        }
    }

    /// Parse multiple bytes.
    pub fn parse(&mut self, data: &[u8]) -> Vec<Action> {
        let mut actions = Vec::new();
        for byte in data {
            actions.extend(self.parse_byte(*byte));
        }
        actions
    }

    /// Reset the parser state.
    pub fn reset(&mut self) {
        self.state = ParserState::Ground;
        self.params.clear();
        self.intermediates.clear();
        self.string_data.clear();
    }

    /// Get current state.
    pub fn state(&self) -> ParserState {
        self.state
    }

    /// Ground state processing.
    fn parse_ground(&mut self, byte: u8) -> Vec<Action> {
        match byte {
            // Control characters
            0x00..=0x1F => self.handle_control(byte),
            // ESC
            0x1B => {
                self.state = ParserState::Escape;
                Vec::new()
            }
            // Normal characters
            _ => vec![Action::Print(byte)],
        }
    }

    /// Escape state processing.
    fn parse_escape(&mut self, byte: u8) -> Vec<Action> {
        match byte {
            // Intermediate bytes (0x20-0x2F)
            0x20..=0x2F => {
                self.intermediates.push(byte);
                self.state = ParserState::EscapeIntermediate;
                Vec::new()
            }
            // CSI
            b'[' => {
                self.state = ParserState::CsiEntry;
                self.params.clear();
                self.intermediates.clear();
                Vec::new()
            }
            // OSC
            b']' => {
                self.state = ParserState::OscString;
                self.string_data.clear();
                Vec::new()
            }
            // DCS
            b'P' => {
                self.state = ParserState::DcsEntry;
                self.params.clear();
                self.intermediates.clear();
                self.string_data.clear();
                Vec::new()
            }
            // SOS
            b'X' => {
                self.state = ParserState::String;
                self.string_data.clear();
                Vec::new()
            }
            // PM
            b'^' => {
                self.state = ParserState::String;
                self.string_data.clear();
                Vec::new()
            }
            // APC
            b'_' => {
                self.state = ParserState::String;
                self.string_data.clear();
                Vec::new()
            }
            // Character set designation
            b'(' | b')' | b'*' | b'+' => {
                // G0-G3 designation follows
                if byte == b'(' {
                    self.active_charset = 0;
                } else if byte == b')' {
                    self.active_charset = 1;
                } else if byte == b'*' {
                    self.active_charset = 2;
                } else {
                    self.active_charset = 3;
                }
                self.state = ParserState::EscapeIntermediate;
                Vec::new()
            }
            // Single character escapes
            b'D' => {
                self.state = ParserState::Ground;
                vec![Action::Index]
            }
            b'E' => {
                self.state = ParserState::Ground;
                vec![Action::NextLine]
            }
            b'H' => {
                self.state = ParserState::Ground;
                vec![Action::SetTabStop]
            }
            b'M' => {
                self.state = ParserState::Ground;
                vec![Action::ReverseIndex]
            }
            b'N' => {
                self.state = ParserState::Ground;
                vec![Action::SingleShiftG2]
            }
            b'O' => {
                self.state = ParserState::Ground;
                vec![Action::SingleShiftG3]
            }
            b'Z' => {
                self.state = ParserState::Ground;
                vec![Action::DeviceAttribute]
            }
            b'\\' => {
                self.state = ParserState::Ground;
                vec![Action::StringTerminator]
            }
            b'7' => {
                self.state = ParserState::Ground;
                vec![Action::SaveCursor]
            }
            b'8' => {
                self.state = ParserState::Ground;
                vec![Action::RestoreCursor]
            }
            b'=' => {
                self.state = ParserState::Ground;
                vec![Action::KeypadApplication]
            }
            b'>' => {
                self.state = ParserState::Ground;
                vec![Action::KeypadNumeric]
            }
            b'c' => {
                self.state = ParserState::Ground;
                vec![Action::ResetTerminal]
            }
            // Unknown escape
            _ => {
                self.state = ParserState::Ground;
                Vec::new()
            }
        }
    }

    /// CSI entry state processing.
    fn parse_csi_entry(&mut self, byte: u8) -> Vec<Action> {
        match byte {
            // Parameter bytes (0x30-0x3F)
            b'0'..=b'9' | b';' | b':' | b'<' | b'=' | b'>' | b'?' => {
                self.params.clear();
                if byte >= b'0'&& byte <= b'9' {
                    self.params.push((byte - b'0') as i64);
                } else {
                    self.params.push(0);
                }
                self.state = ParserState::CsiParam;
                Vec::new()
            }
            // Intermediate bytes (0x20-0x2F)
            0x20..=0x2F => {
                self.intermediates.push(byte);
                self.state = ParserState::CsiIntermediate;
                Vec::new()
            }
            // Final bytes (0x40-0x7E)
            0x40..=0x7E => {
                self.state = ParserState::Ground;
                self.handle_csi(byte)
            }
            // ESC
            0x1B => {
                self.state = ParserState::Escape;
                Vec::new()
            }
            // Control
            0x00..=0x1F => self.handle_control(byte),
            // Invalid
            _ => {
                self.state = ParserState::Ground;
                Vec::new()
            }
        }
    }
}
```

```rust
/// Graphics rendition attribute.
#[derive(Clone, Debug, PartialEq)]
pub enum GraphicsRendition {
    Reset,
    Bold,
    Faint,
    Italic,
    Underline,
    Blink,
    Reverse,
    Hidden,
    Strikethrough,
    Font(u8),
    Foreground16(u8),
    Background16(u8),
    Foreground256(u8),
    Background256(u8),
    ForegroundColor { r: u8, g: u8, b: u8 },
    BackgroundColor { r: u8, g: u8, b: u8 },
    DefaultForeground,
    DefaultBackground,
    UnderlineStyle(Underline),
    NoBold,
    NoItalic,
    NoUnderline,
    NoBlink,
    NoReverse,
    NoHidden,
    NoStrikethrough,
}

impl Action {
    /// Parse SGR parameters into graphics renditions.
    pub fn parse_sgr(params: &[i64]) -> Vec<GraphicsRendition> {
        let mut renditions = Vec::new();
        let mut i = 0;
        
        while i < params.len() {
            let param = params[i].max(0) as usize;
            
            match param {
                0 => renditions.push(GraphicsRendition::Reset),
                1 => renditions.push(GraphicsRendition::Bold),
                2 => renditions.push(GraphicsRendition::Faint),
                3 => renditions.push(GraphicsRendition::Italic),
                4 => renditions.push(GraphicsRendition::Underline),
                5 => renditions.push(GraphicsRendition::Blink),
                7 => renditions.push(GraphicsRendition::Reverse),
                8 => renditions.push(GraphicsRendition::Hidden),
                9 => renditions.push(GraphicsRendition::Strikethrough),
                22 => renditions.push(GraphicsRendition::NoBold),
                23 => renditions.push(GraphicsRendition::NoItalic),
                24 => renditions.push(GraphicsRendition::NoUnderline),
                25 => renditions.push(GraphicsRendition::NoBlink),
                27 => renditions.push(GraphicsRendition::NoReverse),
                28 => renditions.push(GraphicsRendition::NoHidden),
                29 => renditions.push(GraphicsRendition::NoStrikethrough),
                30..=37 => renditions.push(GraphicsRendition::Foreground16(param as u8 - 30)),
                38 => {
                    // Extended foreground color
                    if i + 2 < params.len() && params[i + 1] == 5 {
                        renditions.push(GraphicsRendition::Foreground256(params[i + 2] as u8));
                        i += 2;
                    } else if i + 4 < params.len() && params[i + 1] == 2 {
                        renditions.push(GraphicsRendition::ForegroundColor {
                            r: params[i + 2] as u8,
                            g: params[i + 3] as u8,
                            b: params[i + 4] as u8,
                        });
                        i += 4;
                    }
                }
                39 => renditions.push(GraphicsRendition::DefaultForeground),
                40..=47 => renditions.push(GraphicsRendition::Background16(param as u8 - 40)),
                48 => {
                    // Extended background color
                    if i + 2 < params.len() && params[i + 1] == 5 {
                        renditions.push(GraphicsRendition::Background256(params[i + 2] as u8));
                        i += 2;
                    } else if i + 4 < params.len() && params[i + 1] == 2 {
                        renditions.push(GraphicsRendition::BackgroundColor {
                            r: params[i + 2] as u8,
                            g: params[i + 3] as u8,
                            b: params[i + 4] as u8,
                        });
                        i += 4;
                    }
                }
                49 => renditions.push(GraphicsRendition::DefaultBackground),
                58..=59 => {
                    // Underline color (similar to foreground)
                    i += 1;
                }
                _ => {}
            }
            
            i += 1;
        }
        
        if renditions.is_empty() {
            renditions.push(GraphicsRendition::Reset);
        }
        
        renditions
    }
}
```

| `CSI n d` | VPA | Vertical position absolute | n=1 |
| `CSI n ; m H` | CUP | Cursor position | n=1, m=1 |
| `CSI n ; m f` | HVP | Horizontal vertical position | n=1, m=1 |
| `CSI s` | SCP | Save cursor position | - |
| `CSI u` | RCP | Restore cursor position | - |

## Notes

### Performance Considerations

1. **Zero-copy parsing**: Don't allocate strings unnecessarily
2. **SIMD scanning**: Use `memchr` for ESC detection
3. **Pre-allocated buffers**: Reuse parameter vectors
4. **Batch processing**: Parse multiple bytes at once
