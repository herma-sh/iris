## Quake Mode

### Overview

Quake mode provides a drop-down terminal that appears from the top of the screen:

1. Press hotkey (e.g., F12 or Ctrl+`)
2. Terminal slides down from top
3. Press hotkey again to hide
4. Terminal slides up and disappears

### Benefits

- Quick access without window management
- Doesn't clutter taskbar
- Gaming/overlay-friendly
- Familiar to Quake console users

### Implementation

#### Window Positioning

```rust
struct QuakeConfig {
    enabled: bool,
    hotkey: String,              // e.g., "F12"
    screen_edge: ScreenEdge,   // Top, Bottom, Left, Right
    height: f32,                // Percentage of screen (0.0-1.0)
    width: f32,                 // Percentage of screen (0.0-1.0)
    animation_duration: Duration,
}

enum ScreenEdge {
    Top,
    Bottom,
    Left,
    Right,
}

impl QuakeMode {
    fn calculate_window_rect(&self, monitor: &Monitor) -> Rect {
        let screen = monitor.virtual_work_area();
        
        match self.config.screen_edge {
            ScreenEdge::Top => Rect::new(
                screen.x,
                screen.y,
                screen.width * self.config.width,
                screen.height * self.config.height,
            ),
            ScreenEdge::Bottom => Rect::new(
                screen.x,
                screen.y + screen.height * (1.0 - self.config.height),
                screen.width * self.config.width,
                screen.height * self.config.height,
            ),
            // ...
        }
    }
}
```

#### Animation

```rust
enum QuakeState {
    Hidden,
    Showing { progress: f64, start: Instant },
    Visible,
    Hiding { progress: f64, start: Instant },
}

impl QuakeMode {
    fn update(&mut self, now: Instant) {
        match self.state {
            QuakeState::Showing { start, .. } => {
                let elapsed = now.duration_since(start);
                let duration = self.config.animation_duration;
                
                if elapsed >= duration {
                    self.state = QuakeState::Visible;
                } else {
                    let progress = elapsed.as_secs_f64() / duration.as_secs_f64();
                    self.state = QuakeState::Showing { progress, start };
                }
            }
            QuakeState::Hiding { start, .. } => {
                let elapsed = now.duration_since(start);
                let duration = self.config.animation_duration;
                
                if elapsed >= duration {
                    self.state = QuakeState::Hidden;
                    self.window.hide();
                } else {
                    let progress = 1.0 - (elapsed.as_secs_f64() / duration.as_secs_f64());
                    self.state = QuakeState::Hiding { progress, start };
                }
            }
            _ => {}
        }
    }
    
    fn toggle(&mut self) {
        let now = Instant::now();
        
        match self.state {
            QuakeState::Hidden => {
                self.state = QuakeState::Showing { progress: 0.0, start: now };
                self.window.show();
            }
            QuakeState::Visible => {
                self.state = QuakeState::Hiding { progress: 1.0, start: now };
            }
            _ => {} // Ignore during animation
        }
    }
}
```

#### Always On Top

```rust
impl QuakeMode {
    fn set_always_on_top(&self, on_top: bool) {
        #[cfg(target_os = "windows")]
        {
            use windows::Win32::UI::WindowsAndMessaging::*;
            unsafe {
                let hwnd = self.window.hwnd();
                SetWindowPos(
                    hwnd,
                    if on_top { HWND_TOPMOST } else { HWND_NOTOPMOST },
                    0, 0, 0, 0,
                    SWP_NOMOVE | SWP_NOSIZE | SWP_SHOWWINDOW,
                );
            }
        }
        
        #[cfg(target_os = "macos")]
        {
            use cocoa::appkit::*;
            let window = self.window.ns_window();
            window.setLevel_(if on_top { 
                NSWindowLevel::NSFloatingWindowLevel 
            } else { 
                NSWindowLevel::NSNormalWindowLevel 
            });
        }
        
        #[cfg(target_os = "linux")]
        {
            // X11: _NET_WM_STATE_ABOVE
            // ...
        }
    }
}
```

## Platform Differences

### Windows

- Global hotkey: RegisterHotKey API
- Always on top: SetWindowPos(HWND_TOPMOST)
- Animation: AnimateWindow API or custom

### macOS

- Global hotkey: NSEvent monitoring or CGEventTap
- Always on top: NSWindow.setFloatingPanel(true)
- Animation: NSWindow animator

### Linux

- Global hotkey (X11): XGrabKey
- Global hotkey (Wayland): May require portal support
- Always on top: _NET_WM_STATE_ABOVE
- Always on top (Wayland): May not be supported
