# Phase 2: wgpu Renderer

**Duration**: 2-3 weeks  
**Priority**: P0 Critical  
**Dependencies**: Phase 1  
**Goal**: GPU-accelerated rendering with 60fps scrolling

---

## Overview

This phase implements the GPU renderer using wgpu. The renderer takes the terminal grid and displays it on screen using the GPU. Key features: texture atlas for glyphs, efficient batching, damage-only rendering, and 60fps scrolling even with millions of lines of scrollback.

```rust
/// Renderer configuration.

pub struct Renderer {

/// Cell dimensions.
#[derive(Clone, Copy, Debug)]

    /// Cell width in pixels.
    pub width: f32,
    /// Cell height in pixels.
    pub height: f32,

impl Renderer {
    /// Create a new renderer.
    pub async fn new(
        window: &dyn raw_window_handle::HasRawWindowHandle,
        config: RendererConfig,
    ) -> Result<Self> {
        // Create instance
        let instance = Instance::new(InstanceDescriptor {
            backends: Backends::all(),
            dx12_shader_compiler: Dx12Compiler::default(),
        });
        
        // Create surface
        let surface = unsafe { instance.create_surface(window) }
            .map_err(|e| Error::SurfaceError(e.to_string()))?;
        
        // Request adapter
        let adapter = instance
            .request_adapter(&RequestAdapterOptions {
                power_preference: PowerPreference::HighPerformance,
                compatible_surface: Some(&surface),
                force_fallback_adapter: false,
            })
            .await
            .ok_or(Error::NoAdapter)?;
        
        // Request device
        let (device, queue) = adapter
            .request_device(
                &DeviceDescriptor {
                    label: Some("iris-renderer"),
                    features: Features::empty(),
                    limits: Limits::default(),
                },
                None,
            )
            .await
            .map_err(|e| Error::DeviceError(e.to_string()))?;
        
        let device = Arc::new(device);
        let queue = Arc::new(queue);
        
        // Get texture format
        let format = surface.get_capabilities(&adapter).formats[0];
        
        // Configure surface
        let size = window.inner_size();
        surface.configure(
            &device,
            &SurfaceConfiguration {
                usage: TextureUsages::RENDER_ATTACHMENT,
                format,
                width: size.width,
                height: size.height,
                present_mode: if config.vsync {
                    PresentMode::Fifo
                } else {
                    PresentMode::Immediate
                },
                alpha_mode: CompositeAlphaMode::Auto,
                view_formats: vec![],
            },
        );
        
        // Create font manager
        let font_manager = FontManager::new(&device, &config.font_family, config.font_size)?;
        
        // Calculate cell size
        let metrics = font_manager.metrics();
        let cell_size = CellSize {
            width: metrics.advance_width as f32,
            height: metrics.line_height as f32* config.line_height,
            baseline: metrics.baseline as f32,
        };
        
        // Create atlas
        let atlas = GlyphAtlas::new(&device, 2048, 2048)?;
        
        // Create glyph cache
        let glyph_cache = GlyphCache::new();
        
        // Create pipeline
        let pipeline = RenderPipeline::new(&device, format)?;
        
        // Create buffers
        let vertex_buffer = device.create_buffer(&BufferDescriptor {
            label: Some("vertices"),
            size: 1024 * 1024,  // 1MB
            usage: BufferUsages::VERTEX | BufferUsages::COPY_DST,
            mapped_at_creation: false,
        });
        
        let instance_buffer = device.create_buffer(&BufferDescriptor {
            label: Some("instances"),
            size: 1024 * 1024,  // 1MB
            usage: BufferUsages::VERTEX | BufferUsages::COPY_DST,
            mapped_at_creation: false,
        });
        
        let uniform_buffer = device.create_buffer(&BufferDescriptor {
            label: Some("uniforms"),
            size: std::mem::size_of::<Uniforms>() as u64,
            usage: BufferUsages::UNIFORM | BufferUsages::COPY_DST,
            mapped_at_creation: false,
        });
        
        Ok(Self {
            device,
            queue,
            pipeline,
            glyph_cache,
            atlas,
            font_manager,
            theme: config.theme,
            cell_size,
            grid_size: GridSize::default(),
            damage: Vec::new(),
            vertex_buffer,
            instance_buffer,
            bind_group: atlas.bind_group(),
            uniform_buffer,
        })
    }

    /// Resize the renderer.
    pub fn resize(&mut self, width: u32, height: u32) -> Result<()> {
        // Recalculate grid size
        let cols = (width as f32 / self.cell_size.width) as usize;
        let rows = (height as f32 / self.cell_size.height) as usize;
        self.grid_size = GridSize { rows, cols };
        
        // Mark entire screen as damaged
        self.damage.push(DamageRegion::all());
        
        Ok(())
    }

    /// Update the grid and mark damage.
    pub fn update(&mut self, grid: &[Vec<Cell>], damage: Vec<DamageRegion>) -> Result<()> {
        // Update glyph cache
        for region in &damage {
            for row in region.start_row..=region.end_row.min(grid.len() - 1) {
                forcol in region.start_col..=region.end_col.min(grid[row].len() - 1) {
                    let cell = &grid[row][col];
                    if !cell.is_empty() {
                        // Rasterize and cache glyph
                        let glyph_id = self.font_manager.glyph_id(cell.char);
                        if !self.atlas.contains(glyph_id) {
                            let glyph = self.font_manager.rasterize(cell.char, cell.attrs)?;
                            let rect = self.atlas.allocate(&self.device, glyph.width, glyph.height)?;
                            self.atlas.upload(&self.queue, glyph.data(), rect)?;
                        }
                    }
                }
            }
        }
        
        self.damage.extend(damage);
        Ok(())
    }

    /// Render the terminal.
    pub fn render(&mut self, surface: &Surface) -> Result<()> {
        let frame = surface
            .get_current_texture()
            .map_err(|e| Error::SurfaceError(e.to_string()))?;
        
        let view = frame
            .texture
            .create_view(&TextureViewDescriptor::default());
        
        let mut encoder = self
            .device
            .create_command_encoder(&CommandEncoderDescriptor {
                label: Some("encoder"),
            });
        
        // Clear
        {
            let bg = self.theme.colors.background;
            let _ = encoder.begin_render_pass(&RenderPassDescriptor {
                label: Some("clear"),
                color_attachments: &[Some(RenderPassColorAttachment {
                    view: &view,
                    resolve_target: None,
                    ops: Operations {
                        load: LoadOp::Clear(Color {
                            r: bg.r as f64 / 255.0,
                            g: bg.g as f64 / 255.0,
                            b: bg.b as f64 / 255.0,
                            a: bg.a as f64 / 255.0,
                        }),
                        store: StoreOp::Store,
                    },
                    depth_slice: None,
                })],
                depth_stencil_attachment: None,
                timestamp_writes: None,
                occlusion_query_set: None,
            });
        }
        
        // Build instance data for damaged cells
        // ... (implementation details)
        
        // Submit
        self.queue.submit(std::iter::once(encoder.finish()));
        frame.present();
        
        // Clear damage
        self.damage.clear();
        
        Ok(())
    }

    /// Get cell size.
    pub fn cell_size(&self) -> CellSize {
        self.cell_size
    }

    /// Get grid size.
    pub fn grid_size(&self) -> GridSize {
        self.grid_size
    }

    /// Set theme.
    pub fn set_theme(&mut self, theme: Theme) {
        self.theme = theme;
    }
}
```

```rust
use crate::error::Result;
use wgpu::*;

/// A rectangular region in the texture atlas.
#[derive(Clone, Copy, Debug)]
pub struct AtlasRect {
    pub x: u32,
    pub y: u32,
    pub width: u32,
    pub height: u32,
}

/// Glyph atlas for texture packing.
pub struct GlyphAtlas {
    /// Texture atlas.
    texture: Texture,
    /// Texture view.
    view: TextureView,
    /// Bind group layout.
    bind_group_layout: BindGroupLayout,
    /// Bind group.
    bind_group: BindGroup,
    /// Sampler.
    sampler: Sampler,
    /// Atlas allocator.
    allocator: AtlasAllocator,
}

impl GlyphAtlas {
    /// Create a new glyph atlas.
    pub fn new(device: &Device, width: u32, height: u32) -> Result<Self> {
        let texture = device.create_texture(&TextureDescriptor {
            label: Some("glyph-atlas"),
            size: Extent3d {
                width,
                height,
                depth_or_array_layers: 1,
            },
            mip_level_count: 1,
            sample_count: 1,
            dimension: TextureDimension::D2,
            format: TextureFormat::R8Unorm,
            usage: TextureUsages::TEXTURE_BINDING | TextureUsages::COPY_DST,
            view_formats: &[],
        });
        
        let view = texture.create_view(&TextureViewDescriptor::default());
        
        let sampler = device.create_sampler(&SamplerDescriptor {
            label: Some("glyph-sampler"),
            address_mode_u: AddressMode::ClampToEdge,
            address_mode_v: AddressMode::ClampToEdge,
            address_mode_w: AddressMode::ClampToEdge,
            mag_filter: FilterMode::Linear,
            min_filter: FilterMode::Linear,
            mipmap_filter: FilterMode::Nearest,
            lod_min_clamp: 0.0,
            lod_max_clamp: 0.0,
            compare: None,
            anisotropy_clamp: 1,
            border_color: None,
        });
        
        let bind_group_layout = device.create_bind_group_layout(&BindGroupLayoutDescriptor {
            label: Some("atlas-layout"),
            entries: &[
                BindGroupLayoutEntry {
                    binding: 0,
                    visibility: ShaderStages::FRAGMENT,
                    ty: BindingType::Texture {
                        sample_type: TextureSampleType::Float { filterable: true },
                        view_dimension: TextureViewDimension::D2,
                        multisampled: false,
                    },
                    count: None,
                },
                BindGroupLayoutEntry {
                    binding: 1,
                    visibility: ShaderStages::FRAGMENT,
                    ty: BindingType::Sampler(SamplerBindingType::Filtering),
                    count: None,
                },
            ],
        });
        
        let bind_group = device.create_bind_group(&BindGroupDescriptor {
            label: Some("atlas-bind-group"),
            layout: &bind_group_layout,
            entries: &[
                BindGroupEntry {
                    binding: 0,
                    resource: BindingResource::TextureView(&view),
                },
                BindGroupEntry {
                    binding: 1,
                    resource: BindingResource::Sampler(&sampler),
                },
            ],
        });
        
        let allocator = AtlasAllocator::new(width, height);
        
        Ok(Self {
            texture,
            view,
            bind_group_layout,
            bind_group,
            sampler,
            allocator,
        })
    }

    /// Check if a glyph is in the atlas.
    pub fn contains(&self, id: u64) -> bool {
        self.allocator.contains(id)
    }

    /// Allocate space for a glyph.
    pub fn allocate(&mut self, device: &Device, width: u32, height: u32) -> Result<AtlasRect> {
        self.allocator.allocate(width, height)
            .ok_or_else(|| crate::error::Error::AtlasFull)
    }

    /// Upload glyph data to the atlas.
    pub fn upload(&self, queue: &Queue, data: &[u8], rect: AtlasRect) -> Result<()> {
        queue.write_texture(
            ImageCopyTexture {
                texture: &self.texture,
                mip_level: 0,
                origin: Origin3d {
                    x: rect.x,
                    y: rect.y,
                    z: 0,
                },
                aspect: TextureAspect::All,
            },
            data,
            ImageDataLayout {
                offset: 0,
                bytes_per_row: Some(rect.width),
                rows_per_image: Some(rect.height),
            },
            Extent3d {
                width: rect.width,
                height: rect.height,
                depth_or_array_layers: 1,
            },
        );
        
        Ok(())
    }

    /// Get the bind group.
    pub fn bind_group(&self) -> &BindGroup {
        &self.bind_group
    }
}

/// Simple rectangle packer for the atlas.
struct AtlasAllocator {
    width: u32,
    height: u32,
    /// (x, y) of next allocation.
    next_x: u32,
    next_y: u32,
    /// Row height for current row.
    row_height: u32,
    /// Allocated glyphs.
    allocated: std::collections::HashMap<u64, AtlasRect>,
}

impl AtlasAllocator {
    fn new(width: u32, height: u32) -> Self {
        Self {
            width,
            height,
            next_x: 0,
            next_y: 0,
            row_height: 0,
            allocated: std::collections::HashMap::new(),
        }
    }

    fn contains(&self, id: u64) -> bool {
        self.allocated.contains_key(&id)
    }

    fn allocate(&mut self, width: u32, height: u32) -> Option<AtlasRect> {
        // Try to fit in current row
        if self.next_x + width <= self.width {
            let rect = AtlasRect {
                x: self.next_x,
                y: self.next_y,
                width,
                height,
            };
            self.next_x += width;
            self.row_height = self.row_height.max(height);
            return Some(rect);
        }
        
        // Start new row
        self.next_y += self.row_height;
        self.next_x = 0;
        self.row_height = height;
        
        if self.next_y + height <= self.height {
            let rect = AtlasRect {
                x: 0,
                y: self.next_y,
                width,
                height,
            };
            self.next_x = width;
            return Some(rect);
        }
        
        None
    }
}
```

### shaders/vertex.wgsl

```wgsl
struct VertexInput {
    @location(0) position: vec2<f32>,
    @location(1) tex_coords: vec2<f32>,
}

struct VertexOutput {
    @builtin(position) position: vec4<f32>,
    @location(0) tex_coords: vec2<f32>,
    @location(1) fg_color: vec4<f32>,
    @location(2) bg_color: vec4<f32>,
}

struct Uniforms {
    resolution: vec2<f32>,
    cell_size: vec2<f32>,
    scroll_offset: f32,
    _padding: u32,
}

@group(0) @binding(0) var<uniform> uniforms: Uniforms;

@vertex
fn vertex(input: VertexInput) -> VertexOutput {
    var output: VertexOutput;
    
    // Convert from cell coordinates to normalized coordinates
    let x = (input.position.x * uniforms.cell_size.x) / uniforms.resolution.x * 2.0 - 1.0;
    let y = (input.position.y * uniforms.cell_size.y + uniforms.scroll_offset) / uniforms.resolution.y * 2.0 - 1.0;
    
    output.position = vec4<f32>(x, -y, 0.0, 1.0);
    output.tex_coords = input.tex_coords;
    
    return output;
}
```

### shaders/fragment.wgsl

```wgsl
struct VertexOutput {
    @builtin(position) position: vec4<f32>,
    @location(0) tex_coords: vec2<f32>,
    @location(1) fg_color: vec4<f32>,
    @location(2) bg_color: vec4<f32>,
}

@group(0) @binding(0) var atlas: texture_2d<f32>;
@group(0) @binding(1) var atlas_sampler: sampler;

@fragment
fn fragment(input: VertexOutput) -> @location(0) vec4<f32> {
    // Sample glyph from atlas
    let glyph = textureSample(atlas, atlas_sampler, input.tex_coords);
    
    // Background color
    let bg = input.bg_color;
    
    // Foreground color with glyph alpha
    let fg = input.fg_color;
    
    // Blend foreground over background using glyph alpha
    let result = mix(bg, fg, glyph.r);
    
    return result;
}
```

## Notes

### Design Decisions

1. **R8Unorm texture format**: Single channel for glyph mask, enough for text rendering
2. **Row-based atlas allocation**: Simple, fast, good cache locality
3. **Instance-based rendering**: One instance per cell, reduces vertex data
4. **Damage-only rendering**: Only update changed cells

### Performance Considerations

1. **Glyph caching**: Cache rendered glyphs to avoid re-rasterization
2. **Atlas packing**: Tight packing of glyphs in texture atlas
3. **Batch rendering**: Single draw call for all cells
4. **Uniform buffer**: Update once per frame, not per cell

### Future Optimizations

1. **Multiple atlases**: For unicode ranges, reduce texture size
2. **Distance fields**: For better scaling
3. **Subpixel rendering**: For sharper text
4. **Compute shaders**: For font rendering
